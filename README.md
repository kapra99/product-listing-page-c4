# MyFancyStore - Product Listing Page

A modern, responsive Product Listing Page built with React and Bootstrap 5.

## 🚀 Key Features

- **Dynamic Filtering & Sorting**: Browse products by category, color, and price.
- **Product Detail View**: Full-page view for every product with detailed information.
- **Bootstrap 5 Toasts**: Premium notification system for cart actions.
- **Load More Pagination**: Optimized performance with progressive data loading.

## 🛠️ Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Run the Development Server

```bash
npm run dev
```

### 3. Generate Mock Data (Optional)

If you need to regenerate the product list:

```bash
node generate_products.js
```
